# Smart College ERP System (Java)

## 📌 Overview
This project is a Smart College ERP System developed using Java Swing.  
It is a role-based application designed to manage student-related academic and administrative data.

The system allows:
- Faculty to manage student records
- Students to view their personal dashboard

It includes features like persistent storage, analytics, reports, and smart alerts.

---

## 🚀 Features
- 🔐 Role-based Login (Faculty / Student)
- 👨‍🎓 Student Management System
- 📊 Attendance, Marks, Fee Tracking
- 📈 Student Dashboard (GPA, Attendance, Status)
- ⚠️ Risk Prediction & Suggestions
- 🔔 Notification System
- 📑 Report Generation & Export
- 💾 File-based Persistent Storage
- 🎨 Responsive Java Swing GUI

## ▶️ How to Run the Project

### Step 1: Open Project
- Open the project folder in VS Code or any Java IDE

### Step 2: Go to src Folder
Make sure you are inside the `src` directory

### Step 3: Compile
Open terminal and run:
javac Main.java ui/.java service/.java model/*.java

### Step 4: Run
java Main
